Example for the showsource library

The purpose for this library es to extract the code from a existing portion of a web page to show it as a new part in the web page.

(*) The origin of the library is to ease the creation of examples of existing libraries.

Simple example

hint: add the data-showsource attribute to the element that we want to extract the html content and to add the result after it.

the default behavior is to hide the root element (in this case, data-showsource)

Extended simple example

This is almost the same example than before, but we are overridding some default options.

  • For the shake of better reading the examples, the default behavior is to remove tags like h1, h2, h3, h4, h5, h6, p. In this case we are not hidding them, by overridding data-showsource-remove
  • We want to show the data attributes related to this plugin, so we added data-showsource-no-hide-plugin.
